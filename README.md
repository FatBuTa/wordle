## Getting Started

First, install dependencies:

```bash
npm i
```

Second, build:

```bash
npm run build
```

Start server

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

test:
```bash
npm run cypress
#note: run local server before test
```

development
```bash
npm i
npm run dev
npm start
```
